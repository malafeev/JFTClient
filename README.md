## JFTClient


JavaFX based ssh client. 


## Building from Source

### Prerequisites

- JDK 8
- Gradle 
- rsync & sshpass

### Supported OS

- Linux
- OS X

### Installation

In the directory that contains the build.gradle run

     gradle install
     
It will build, test and install application to *~/jftclient* folder.
 

### To run

Start JFTClient: 

    ~/jftclient/bin/jftclient
